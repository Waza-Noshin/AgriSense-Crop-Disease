# 🌾 AgriSense - Crop Disease Detection

## Project Overview
AI-powered mobile web app that helps farmers detect crop diseases by uploading a photo of the affected plant.

## Social Impact
- 58% of Indians depend on agriculture
- Late disease detection destroys 30-40% of crops
- Farmers can't afford expert visits

## Solution
- Farmers upload crop photos via mobile
- AI detects disease within seconds
- Treatment recommendations in Hindi & English

## Tech Stack
| Component | Technology |
